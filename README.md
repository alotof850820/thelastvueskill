# vue-first-app

## 目的

```
使用Vue3.0以上之語法<script setup>及composition API 搭配 pinia 和 vuecli 建構小型App
```

### 描述

```
製作包含component pages route auth 及數據流動 和動畫
```

### 功能

```
user 可以註冊(signup)後再登入系統，登入系統可以創立個人名牌，且可以接收來自不同user的信息
一小時候會自動登出
```
